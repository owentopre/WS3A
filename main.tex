    Simultaneity proved through commutation,

    \begin{align}
        [\hat{N}_A,\hat{N}_B] &= \hat{a}_{A}^{\dagger}\hat{a}_{A}\hat{a}_{B}^{\dagger}\hat{a}_{B}-\hat{a}_{B}^{\dagger}\hat{a}_{B}\hat{a}_{A}^{\dagger}\hat{a}_{A} \\
        \hat{a}_{\alpha} &= \sqrt{m\omega/2\hbar}(\hat{x}_{\alpha}+i\hat{p}_{\alpha}/m\omega)\\
        \hat{a}_{\alpha}^{\dagger} &= \sqrt{m\omega/2\hbar}(\hat{x}_{\alpha}-i\hat{p}_{\alpha}/m\omega)\\
        \hat{a}_{\alpha}^{\dagger}\hat{a}_{\alpha} &= m\omega/2\hbar(\hat{x}_{\alpha}^{2}+\hat{p}_{\alpha}^{2}/(m\omega)^2)\label{eq:V}
    \end{align}
    And because $[\hat{x}_A, \hat{x}_B]=[\hat{p}_A, \hat{p}_B]=0$, and $\hat{a}_{A}$ and $\hat{a}_B$ depend only on $\hat{x}_A,\hat{p}_A$ and $\hat{x}_B,\hat{p}_B$ respectively, we can treat $\hat{a}_{B}^{\dagger}\hat{a}_{B}\hat{a}_{A}^{\dagger}\hat{a}_{A}$ as $\hat{a}_{A}^{\dagger}\hat{a}_{A}\hat{a}_{B}^{\dagger}\hat{a}_{B}$. Therefore:
    \begin{align}
        [\hat{N}_A, \hat{N}_B] &= 0
    \end{align}
    As $\hat{N}_A$ and $\hat{N}_B$ commute, they are simultaneous and completely independent, so $\hat{N}_A$ depends only particle A and $\hat(N)_B$ on B. This means that we can find numbers for the eigenstates of each operator:
    \begin{align}
        \hat{N}_A\ket{n_A,n_B} = n_A\ket{n_A,n_B}\\
        \hat{N}_B\ket{n_A,n_B} = n_B\ket{n_A,n_B}
    \end{align}
